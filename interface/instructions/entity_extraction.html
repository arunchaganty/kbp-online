First of all, welcome :)

The focus of this task is to find mentions of entities in the given text. Entities can be people, organizations, places, dates or even titles given to people. In these instructions, we talk about common mistakes and have an optional instructional video to help you understand the task better. Remember you can always come back to this page by clicking on in the interface.

Steps

  1. As you read through the text, try to identify named entities (e.g. Carrie Fisher) or any references to named entities (e.g. the star), as well as professional titles (e.g. actress) and dates (e.g. Thursday). You do not need to identify unnamed groups of people, e.g. "Three policemen". We've bolded spans which we guess might be useful, but we aren't that good and miss out on stuff.
  2. Select mentions using your mouse. Different types of entity should be selected separately, so in the phrase "Rev. Graylan Hagler", you should select Rev. (a title) separately from Graylan Hagler (a person).
  3. If you haven't seen this entity in the text before, select its type via the Add new entity pane. You may then be taken to a dialogue box for further processing. Only named people or organizations should be added as new entities (e.g. you should highlight "Shell Oil" but not "Many oil companies".)
  4. If you have seen this entity before then select it from Select entities to link panel. Indirect references to entities should be selected, (e.g. "The company").
  5. Because documents vary in length, we price each document differently to try to pay you fairly. This amount and the expected time the document will take is listed on the top right.

Correcting mistakes

  • If you feel you've made a mistake you can click on the incorrect span, the entity that you had selected would be highlighted.
  • If it was only a matter of you having selected a wrong pre-existing entity, it can be fixed by selecting the right one this time.
  • Otherwise, you can remove the span and redo the process correctly this time

Common mistakes

  1. Creating a new entity instead of selecting an existing one. To fix this mistake, click on the erroneous span and choose the existing entity from the list on the right.
  2. Missing pronouns (e.g. she).
  3. Including people's titles in with the name, e.g. selecting "Speaker Paul Ryan" instead of "Speaker" and "Paul Ryan" separately.
  4. Not looking at mentions that are not bolded.

What is a title?

Official or unofficial name(s) of the employment or membership positions that is held by someone. Here are a few examples

Correct titles

  • Doctor, Lawyer, Founder
  • Chief Neurosurgeon, Associate Partner (the more specific, the better ;)

Incorrect titles

  • Hero, Socialite
  • " reporter with the LA Times" (no employer info required, just reporter is cool)
