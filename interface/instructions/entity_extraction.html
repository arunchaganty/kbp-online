First of all, welcome :)

The focus of this task is to find mentions of entities in the given text. Entities can be people, organizations, places, dates or even titles given to people.

Steps

  1. As you read through the text, see if a word or a span of words strikes you as mentioning an entity (even if it is a pronoun, or referring indrectly e.g. "The president said ...").
  2. Upon finding a mention select it using your mouse (Even clicking on single words works ;)). We've bolded spans which we guess might be useful.
  3. If you've never seen this entity in the text before, select its type via the Add new entity pane
  4. Then you will be optionally taken to a linking dialogue box where the task is to pick the matching Wikipedia entry if possible
  5. If you have seen this entity then select the correct one from Select entities to link panel
  6. If you feel you've made a mistake you can click on the incorrect span and remove the span. The entity that you had selected would be highlighted. If it was only a matter of you having selected a wrong pre-existing entity, it can be fixed by selecting the right one this time.

Common mistakes

  1. It sometimes so happens that you end up creating a new entity instead of selecting one which we had already found.
  2. Pronouns are hard to miss.
  3. Titles of people and dates are easy to miss too.
  4. Our systems aren't great and sometimes we'll miss out on spans to bolden. Please teach us how to do it better.

What is a title?

Official or unofficial name(s) of the employment or membership positions that is held by someone. Here are a few examples

Correct titles

  • Doctor, Lawyer, Founder
  • Chief Neurosurgeon, Associate Partner (the more specific, the better ;)

Incorrect titles

  • Hero, Socialite
  • " reporter with the LA Times" (no employer info required, just reporter is cool)
